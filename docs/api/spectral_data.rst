Spectral Data
=============

Data structures for handling spectral information used in color transformations.

.. contents:: Contents
   :local:
   :depth: 2

Overview
--------

The spectral data module provides structures for representing:

- Camera spectral sensitivity curves
- Illuminant spectral power distributions
- Color matching functions (CMFs)
- Training spectral data
